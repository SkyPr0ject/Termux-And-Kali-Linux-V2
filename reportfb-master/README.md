# reportfb



tutorial cara install auto report akun facebook dengan termux


apt update && apt upgrade
pkg install python2
pkg install git
pkg install figlet
pkg install unzip
pip2 install mechanize

kemudian kita unduh script auto report dengan kode perintah di bawah ini
 git clone https://github.com/kumpulanremaja/report
cd report
setelah itu kita menjalankan script auto report facebook dengan kode perintah
 python2 report.py
